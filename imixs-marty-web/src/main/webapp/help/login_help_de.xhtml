Sie haben noch keine eigene OpenID?

Über eine OpenID können Sie sich auf unterschiedlichen Websites über nur eine URL anmelden. Sie müssen also kein separates Passwort für diese Anwendung verwenden bzw. sich merken. Wen Sie noch keine OpenID besitzen, können Sie über die Auswahlbox neben dem Eingabefeld einen OpenID Anbieter auswählen und dort eine neue OpenID registrieren.
